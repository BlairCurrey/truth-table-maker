# What

A propositional logic language with a C-style syntax built using [ohm](https://github.com/ohmjs/ohm).

# Syntax

1. Logical AND: `&&`
   Example Expression: `P && Q`

2. Logical OR: `||`
   Example Expression: `P || Q`

3. Logical NOT: `!`
   Example Expression: `!P`

4. Group: `()`
   Example Expression: `(P && (Q || R))`
